# Portfolio Rodrigo Gonzalez

Portfolio pessoal com sistema de gerenciamento de projetos baseado em banco de dados SQL.

## 🚀 Funcionalidades

- ✅ Exibição de até 6 projetos por página
- ✅ Paginação entre projetos
- ✅ Página de detalhes para cada projeto
- ✅ Armazenamento em banco de dados SQL
- ✅ API REST para gerenciamento de projetos
- ✅ Preparado para área administrativa (futura implementação)

## 📋 Pré-requisitos

- PHP 7.4 ou superior
- MySQL 5.7 ou superior
- Servidor web (Apache, Nginx, ou PHP built-in server)

## 🔐 Próximos Passos (Área Administrativa)

O projeto está preparado para receber uma área administrativa. A tabela `users` já está criada no schema.

Funcionalidades futuras:
- Login/autenticação
- CRUD completo de projetos
- Upload de imagens
- Editor de markdown
- Gerenciamento de usuários
